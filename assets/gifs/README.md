# GIFs

Poné acá tus gifs de demostración (`.gif`, o también `.webp`/`.mp4` si preferís).

El nombre debe coincidir con el campo `gif` en `data/moves.js`. Por ejemplo:

```js
gif: "assets/gifs/kamehameha.gif"
```

Mientras no exista el archivo, la tarjeta muestra el emoji del movimiento.

**Tip:** para grabarte y convertir a gif liviano, un clip de 2-4 s a ~480p suele alcanzar.
